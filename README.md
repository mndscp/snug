# Snug

Snug is an alternate keyboard layout optimized for usage in both English and German.

### Matrix

```
q l d m b y f o u '
s r t c g p n e i a
z x k w v j h / , .
```
### Row-stagger

```
q l d m b y f o u '
 s r t c g p n e i a
  z x k w v j h / , .
```

### Row-stagger with angle mod

```
q l d m b y f o u '
 s r t c g p n e i a
  x k w v z j h / , .
```
